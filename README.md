# Number Guessing Game (Java)

## Description
This is a simple console-based Number Guessing Game written in Java.  
The program generates a random number between 1 and 100.  
The user have a 10 chances to guess the correct number.

## Features
- Random number generation
- 10 attempts limit
- Hint messages (Greater / Lesser)

## Technologies Used
- Java
- Math.random()

## How to Run

1. Compile the program:
   javac NumberGuessing.java

2. Run the program:
   java NumberGuessing
